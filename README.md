# Grid and Groves

## 🚀 快速开始

### 环境要求

- Godot Engine 4.6 或更高版本
- .NET SDK 8.0+

### 运行项目

1. 克隆仓库
```bash
git clone <repository-url>
cd GridAndGroves
git lfs pull
```

2. 使用 Godot 编辑器打开项目根目录

3. 在 Godot 编辑器中点击运行按钮（F5）
